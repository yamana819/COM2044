# Object-Oriented Programming (Java Language)

This repository contains object-oriented algorithms and laboratory works implemented in Java for the university Object-Oriented Programming (OOP) course.

## 📚 Resources
The main textbook used is **"Java How To Program" by Paul Deitel & Harvey Deitel**. Many examples in this folder are solutions or adaptations of the chapter exercises from this book.

## 📂 Structure
* **`exercises/`**: Chapter-by-chapter solutions to problems found in the Deitel & Deitel textbook.
* **`lab/`**: Practical applications and problems solved during university laboratory sessions.

## 🧠 Key Concepts
This repository includes practices on:
* **OOP Fundamentals:** Classes, objects, constructors, and encapsulation (access modifiers).
* **Inheritance & Polymorphism:** Superclasses, subclasses, method overriding, and interfaces.
* **Core Java:** Exception handling (`try-catch`), arrays, and basic data structures (like `ArrayList`).
* **Object-Oriented Design:** Writing modular, reusable, and clean Java code.

## ⚙️ Compilation & Execution
The codes are developed and tested using **Eclipse Temurin JDK 21.0.11**.
